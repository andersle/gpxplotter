The |gpxplotter| package
========================

This is the documentation for the source code of
the |gpxplotter| library, version |version|.

|gpxplotter| modules
--------------------

.. toctree::
   :hidden:
   :maxdepth: 2

   gpxplotter.common

The |gpxplotter| package is structured into modules as follow:

* :ref:`gpxplotter.common <api-common>` with common methods
  for the plotting and gpx-reading methods.

* :ref:`gpxplotter.folium\_map <api-folium>` with methods
  for generating HTML maps
  with `folium <https://python-visualization.github.io/folium/>`_.

* :ref:`gpxplotter.gpxread <api-gpxread>` with method for
  reading information from gpx-files.

* :ref:`gpxplotter.mplplotting <api-mplplotting>` with methods
  for generating pre-defined plots
  with `matplotlib <http://matplotlib.org/>`_.

* :ref:`gpxplotter.version <api-version>` with version
  information about |gpxplotter|.
